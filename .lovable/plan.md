

# grammar-chat 프롬프트 수정 — "패턴 피드백" 해석 오류 방지

## 문제

스크린샷에서 사용자가 "주격 관계대명사 패턴을 따르라"고 지적하면, AI가 문법 분류 자체를 바꿔버림 (관계대명사 → 명사절 주어). "패턴을 따르라" = "스타일만 바꿔라"인데, AI는 "문법을 바꿔라"로 해석.

## 수정 내용

`supabase/functions/grammar-chat/index.ts`의 `trustAddendum` 수정:

현재:
- "사용자가 '틀렸어' 등으로 오류를 지적하면 다시 분석하라"

추가할 지시:
- 사용자가 "패턴", "고정 패턴", "양식", "형식", "말투"를 언급하며 수정을 요청하면 → **문법 분류(관계대명사/동격/분사 등)는 절대 바꾸지 말고**, 설명의 문체·구조·종결만 고정 패턴에 맞추라
- 문법 분류를 바꾸는 건 오직 사용자가 "이건 관계대명사가 아니라 접속사야" 같이 **구체적 문법 용어로 명시할 때만** 허용

## 수정 파일

| 파일 | 변경 |
|------|------|
| `supabase/functions/grammar-chat/index.ts` | `trustAddendum` 텍스트에 패턴 피드백 해석 규칙 추가 (lines 340-348) |

## 기대 결과
- "패턴 따라" → 문법 분류 유지, 스타일만 교체
- "이건 접속사야" → 문법 분류 변경 허용
- 앵무새 반복 방지는 기존 지시 유지
