

# 직역/의역 숨김 후 복원 기능

## 현재 상태
- X 버튼 클릭 시 `hideLiteral: true` / `hideNatural: true`로 숨김 처리 (삭제 아님)
- 하지만 다시 보이게 하는 버튼이 없음

## 수정: `src/pages/Index.tsx`

직역 또는 의역이 숨겨진 경우, 해당 위치에 작은 복원 버튼을 표시:

```text
┌────────────────────────────────┐
│  [직역 보기] [의역 보기]        │  ← 숨겨진 항목만 표시
└────────────────────────────────┘
```

### 구현
- `result.hideLiteral === true`일 때, 직역 영역 자리에 "직역 보기" 버튼 렌더링
- `result.hideNatural === true`일 때, 의역 영역 자리에 "의역 보기" 버튼 렌더링
- 클릭 시 `hideLiteral: false` / `hideNatural: false`로 복원
- 스타일: 작고 눈에 띄지 않는 텍스트 버튼 (Eye 아이콘 + 라벨)

1개 파일만 수정.
