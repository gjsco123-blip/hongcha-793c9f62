

# WORKBOOK 외곽선 거리 — R을 기준으로 통일

## 변경 요약

`LETTER_METRICS`에서 **R (인덱스 2, inset: 0)**을 기준점으로 삼고, 나머지 7글자의 inset을 R과 동일한 외곽선 거리가 되도록 재조정.

현재 R의 inset은 이미 0이고 K도 0이므로 기준 자체는 동일하지만, 코멘트와 의도를 R 기준으로 명확히 하고 다른 글자들의 값을 미세 조정.

## 현재 → 변경

```text
인덱스  글자  구간           현재 inset  → 새 inset   비고
  0     W    상단 직선       0.2          0.15       R과 비슷한 직선 구간, 약간만 보정
  1     O    상단 직선       0.7          0.5        과보정됨 → 줄여서 R과 맞춤
  2     R    상단→곡선       0            0          ★ 기준점
  3     K    곡선            0            0          R과 유사한 직선 글자
  4     B    곡선→우측       0.55         0.4        과보정 → 줄임
  5     O    우측 직선       0.65         0.5        과보정 → 줄임
  6     O    우측 직선       0.65         0.5        과보정 → 줄임
  7     K    우측 직선       0            0          R과 동일
```

핵심: 둥근 글자(O, B)의 inset이 너무 컸던 게 문제. R 기준으로 맞추면 전체적으로 0.15~0.2pt씩 줄어듦.
