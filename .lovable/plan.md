

# WORKBOOK 곡선 텍스트 — 글자 간격 넓히기 + 외곽선 밀착

## 문제
1. 글자 간격이 너무 좁음
2. "BOOK" 부분이 외곽선에서 떠 있음 (단순 원호로는 직선→곡선→직선 경로를 못 따라감)

## 해결 방향

현재 단순 원호(arc) 방식 대신, body 외곽선의 **실제 경로**(상단 직선 → 18pt 곡선 → 우측 직선)를 따라 글자를 배치하는 **경로 기반(path-based) 방식**으로 전환.

### `getArcLetters()` 완전 재작성

**경로 정의:**
```text
구간 1: 상단 직선 (왼→오) — y 고정, x 증가
구간 2: 곡선 (18pt borderRadius) — 90° 호
구간 3: 우측 직선 (위→아래) — x 고정, y 증가
```

**배치 로직:**
1. 전체 경로 길이 계산 (직선1 + 호 + 직선2)
2. 글자 간격 = 전체 길이 / (글자 수 + 1) — 양 끝 여백 포함
3. 각 글자를 경로 위 등거리 지점에 배치
4. 직선 구간: 회전 0° (상단) 또는 90° (우측)
5. 곡선 구간: 해당 각도의 접선 방향으로 회전

**간격 조절:**
- 경로 시작/끝 위치를 조절하여 글자가 분포하는 범위 제어
- 글자 간 거리를 pt 단위로 직접 설정 가능 (현재 각도 기반보다 직관적)

### 스타일
- fontSize: 6.5pt 유지
- 외곽선과의 거리(offset): border 두께(0.6pt) + 글자 높이 절반(~3pt) ≈ **4pt**
